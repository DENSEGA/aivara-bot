# AIVARA Bot v4.0

**Telegram-бот для компании «ЭкоКаркас»** — строительство каркасных домов ИЖС.

## Что нового в v4.0

### 🏗 Карточки объектов + этапы + прогресс
- Визуальные карточки с прогресс-баром для каждого объекта
- 9 этапов стройки: Фундамент → Каркас → Утепление → Кровля → Фасад → Окна → Инженерка → Отделка → Сдача
- Статусы: ⏳ Ожидание → 🔨 В работе → ✅ Выполнен
- Даты начала/завершения автоматически
- Комментарии к этапам

### 🎨 Усовершенствованные рендеры
- Точный расчёт пропорций дома по этажности (1/1.5/2 этажа)
- Каждый параметр сметы напрямую влияет на промт DALL-E
- Character bible по 16 реальным рендерам ЭкоКаркас
- Детализация: окна, двери, терраса, фасад — всё из конфигуратора

### 👥 Два бота — одна система
- **Admin-бот** (@aivaradens_bot) — для admin + partner
- **Team-бот** (@AIVARATeambot) — для foreman, supply, worker (в разработке)

### 🔑 5 ролей
- `admin` — полный доступ
- `partner` — финансы, сметы, просмотр
- `foreman` — прораб (объекты, этапы, фото-отчёты) — Team-бот
- `supply` — снабженец (заявки, закупки) — Team-бот
- `worker` — рабочий (только задачи) — Team-бот

## Google Sheets — новый лист `stages`

**Создай лист `stages`** в таблице с заголовками:

| A | B | C | D | E | F | G | H |
|---|---|---|---|---|---|---|---|
| Объект | Этап | Статус | Прогресс% | Назначен | Дата начала | Дата завершения | Комментарий |

Этапы создаются автоматически при первом просмотре объекта.

## Деплой

1. Добавить `TELEGRAM_TEAM_BOT_TOKEN` в Railway Variables
2. Создать лист `stages` в Google Sheets (заголовок: A-H)
3. Загрузить файлы на GitHub → Railway автодеплой
