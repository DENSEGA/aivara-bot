# AIVARA Bot v3.3 🌟

Telegram-бот для управления строительным бизнесом **ЭкоКаркас**.

## Что нового в v3.3

### 🏠 Конфигуратор домов (полная переработка)
- **3 типа домов:** Модули / Стандарт / Стандарт+
- **Сваи:** винтовые (89мм, 108мм) и ЖБ (150×150, 200×200 / 3м, 4м)
- **Терраса:** открытая/закрытая + ограждение (п.м.) + ступени
- **Внутренняя отделка:** стены (гипсокартон/вагонка/штукатурка/покраска), потолок (вагонка/натяжной), пол (ламинат/линолеум/кварцвинил/плитка), межкомнатные двери (эконом/стандарт/премиум)
- **Инженерные системы:** отопление, сантехника, электрика, вентиляция (каждый — да/нет)
- **Коммуникации:** септик (да/нет), скважина (да/нет)
- **DALL-E рендеры** — 2 визуализации (вид спереди + сзади), встраиваются в PDF
- **Доп. скидка** — произвольный %, 0 = не отображается

### 💰 Финансы
- Пошаговый ввод расходов: объект → категория → сумма → комментарий
- Отчёты за день / месяц / всё время (короткий + полный формат)
- Список объектов из Google Sheets (лист `objects`)
- Добавление новых объектов (для admin)

### 🤖 ChatGPT
- AI-ассистент на базе GPT-4o-mini
- Голосовой ввод (Whisper)
- Генерация изображений (DALL-E 3) — команда `/image`
- История диалога (до 30 сообщений)

### 📓 Дневник
- Текстовые заметки с сохранением в Google Sheets (лист `daylog`)
- Напоминания с парсингом дат (`завтра 9:00`, `через 2 часа`, `25.03.2026 14:00`)
- Cron-планировщик проверяет каждую минуту

### 👥 Задачи
- Постановка задач команде с дедлайнами
- Уведомления при назначении и выполнении
- Просмотр своих и всех задач

### ⚙️ Роли и доступ
- **admin** — полный доступ ко всем модулям
- **partner** — финансы, смета, ChatGPT, дневник, задачи (без управления)
- **worker** — только задачи

### 🔧 Технические улучшения
- **Фикс `parseFloat` пробелов** — функция `num()` убирает пробелы и запятые из цен
- **Фикс шрифтов Oblique** — fallback на обычные DejaVu (без ошибок в Docker)
- **Dockerfile** с `fonts-dejavu-core`, `fonts-dejavu-extra`, Python + reportlab
- **Прайс с колонками G/H** — разделение цен для Стандарт и Стандарт+
- **PDF** — салатовая тема, шапка белая/зелёная, скидка директора 2%, подарки (бытовка + туалет), цитата с фото руководителя

## Структура проекта

```
aivara-bot/
├── Dockerfile              # Сборка для Railway (Node 20 + Python + шрифты)
├── Procfile                # web: node src/index.js
├── package.json            # v3.3.0
├── .env.example            # Шаблон переменных
├── assets/
│   ├── logo.jpg            # Логотип ЭкоКаркас
│   └── director.jpg        # Фото руководителя для PDF
└── src/
    ├── index.js            # Точка входа, маршрутизация
    ├── modules/
    │   ├── config.js       # Конфигуратор домов + расчёт сметы
    │   ├── finance.js      # Финансы + отчёты
    │   ├── chatgpt.js      # AI-ассистент + голос + картинки
    │   ├── diary.js        # Дневник + напоминания
    │   └── tasks.js        # Задачи команды
    └── services/
        ├── sheets.js       # Google Sheets API (объекты, финансы, прайс, дневник)
        ├── openai.js       # OpenAI API (GPT, DALL-E, Whisper, рендеры)
        ├── pdf.js          # Генерация PDF-сметы через Python/reportlab
        ├── roles.js        # Система ролей (admin/partner/worker)
        └── reminders.js    # Cron-планировщик напоминаний
```

## Google Sheets — листы

| Лист | Назначение |
|------|-----------|
| `objects` | Список строительных объектов (колонка A) |
| `finance` | Записи расходов (timestamp, тип, сумма, объект, категория, комментарий) |
| `price` | Прайс-лист: A — №, B — название, C — ед., D — цена, G — Стандарт, H — Стандарт+ |
| `coefficients` | Коэффициенты расчёта (высота стен, коэфф. крыш и т.д.) |
| `blank` | Реквизиты компании (телефон, email, ИНН, адрес, слоган, сайт) |
| `daylog` | Дневник + напоминания (timestamp, текст, время напоминания) |

## Деплой на Railway

1. Push в GitHub: `DENSEGA/aivara-bot`
2. Railway подхватит `Dockerfile` автоматически
3. Заполнить переменные окружения в Railway Dashboard
4. Бот запустится: `✅ AIVARA Bot v3.3 запущен!`
